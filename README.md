# DEPI-Financial Sample

This repository showcases an end-to-end data analytics project developed as part of my training journey in Data Analysis using Microsoft Power BI. The project is built on a comprehensive financial dataset (Financial Sample.xlsx) containing global sales data across multiple segments, countries, and product lines from 2013-2014.

---
**🎯 Project Objectives**

- Apply data cleaning and transformation skills using Power Query
- Build an efficient relational Data Model (Star Schema)
- Design an interactive Dashboard for financial performance monitoring
- Extract actionable KPIs to support data-driven decision making

-----
**🛠️ Tools & Technologies Used**

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel
